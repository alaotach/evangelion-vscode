# ◤ Evangelion UI — VS Code ◥

An extension that turns VS Code into a **NERV command terminal** with multiple themes.

---

## What this actually does

* **MAGI Command Center (Sidebar)**
  You get a dedicated panel styled after the MAGI system—**MELCHIOR, BALTHASAR, CASPER**.
  Open it from the NERV icon in the activity bar.

* **Angel Detection System**
  > Your editor reacts if you make some mistake.

* **Instant Theme Switching**
  You can switch themes directly from the magi command panel.

* **BGM**
  Subtle instrumental music of *A Cruel Angel’s Thesis*.

## How to use it

1. Open the **NERV panel** from the sidebar
2. Pick your theme inside the MAGI interface
3. Adjust audio
4. Start working
